## [v0.3.0] - 2026-03-06

### 新增功能 (Added)
* 新增 `apps/maze_android/` Compose 客户端，接入 `MazeSession` 事件流、节流消费与实时 `Canvas` 可视化。
* 新增 `libs/maze_export/` 独立导出组件，支持从 `SearchFrame` 序列离线导出 PNG。
* 新增 `apps/maze_win_gui/src/event/`、`apps/maze_win_gui/src/render/`、`apps/maze_win_gui/src/runtime/`、`apps/maze_win_gui/src/state/`，补齐 Windows GUI 事件流渲染链路。
* 新增 `tests/contracts/`、`test/solver_event_emitter_test.cpp`、`test/solver_result_builder_test.cpp` 与 Win GUI 相关测试，覆盖事件顺序、结果构建与表现层解析。

### 技术改进/重构 (Changed/Refactor)
* 重构搜索主链路为事件流模式，统一使用 `ISearchEventSink`、`SolveOptions` 与增量 `SearchEvent` 作为核心契约。
* 重构 `libs/maze_api_c/api/maze_api.h` 为 `MazeSession + MazeEventCallback` 同步会话模型，并让 Win/Android 同步切换到新接口。
* 重构 `apps/maze_win_gui/` 与 `apps/maze_android/` 为本地增量渲染模式，由拓扑快照、单元格增量与路径事件驱动 UI。
* 重构构建目标，将导出能力从 `maze_infra` 主路径中拆分为独立的 `maze_export` 组件。
* 调整内核版本源文件，从 `libs/maze_core/VERSION` 切换到 `libs/maze_core/MAZE_CORE_VERSION.txt`。

### 修复 (Fixed)
* 修复事件流模式下 Win GUI 与 Android 客户端的取消、进度、摘要与错误状态可达性。
* 修复 Android Gradle 迁移到 AGP `9.0.1` 时与 built-in Kotlin 冲突的问题，改为仅保留 `org.jetbrains.kotlin.plugin.compose`。
* 修复 Windows GUI 在高频搜索事件下的整帧重绘开销，改为优先执行 dirty-cell 增量刷新。

### 弃用/删除 (Deprecated/Removed)
* 删除旧 `MazeHandle`、`MazeRunFromJson`、`MazeSetFrameCallback`、`MazeGetFrameCount` 与 `MazeGetFrameRgba` 主路径。
* 删除 CLI、Win GUI 与 Android 运行主路径中的 `write_png` / `output_dir` 写盘耦合，改为显式使用 `libs/maze_export/` 离线导出。
